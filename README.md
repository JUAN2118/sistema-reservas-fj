# Sistema Integral de Reservas Software FJ

Este proyecto consiste en el desarrollo de un sistema en Python basado en Programación Orientada a Objetos (POO), sin el uso de bases de datos.

## Funcionalidades
- Gestión de clientes
- Gestión de servicios (salas, equipos, asesorías)
- Sistema de reservas
- Manejo de excepciones avanzadas
- Registro de eventos y errores en archivo log

## Tecnologías usadas
- Python
- Programación Orientada a Objetos
